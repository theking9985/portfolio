Projects

Real Estate Game

Real Estate Game is a MEAN stack web application using MongoDB, AngularJS, Node, Express, Animate CSS, Sweet Alert, Bootstrap, Javascript, jQuery, HTML5 and CSS3.

User is given $1.000.000 to buy and sell houses. Random events occur affecting house price.

ParkIt

ParkIt is a full stack web application using Ruby on Rails, Google API, Stripe API, PostgreSQL, SQL, Bootstrap, Javascript, jQuery, HTML5 and CSS3.

User can rent a parking spot from someone or make money by renting out their own parking spot.

Calories counter

Calories counter is a full stack web application using Node, Express, PostgreSQL, SQL, USDA API, JavaScript, jQuery, Bootstrap, Font Awesome, HTML5 and CSS3.

User sets daily goal and logs their calories intake by choosing food products from USDA API.

Blackjack

Blackjack is a front-end game using JavaScript, jQuery, Bootstrap, Sweet Alert, HTML5 and CSS3.

User starts with $1000 and plays blackjack headsup with casino.
